16. 修改课程状态 上架下架
URL = course/updateStatus
请求方式：POST
参数类型：x-www-form-urlencoded
请求参数：
	courseId:课程id
	status:状态 true-上架 false-下架
返回参数：
{
    "code": 0,
    "message": "修改成功",
    "totalPage": null,
    "totalElement": null,
    "data": null
}

17. 删除课程
URL = course/delete/{type}/{id}
请求方式：GET
参数类型：路径参数
请求参数：
	type:0-课程 1-课时id
	id:删课程时传课程id，删课时时传课时id
返回参数：
{
    "code": 0,
    "message": "删除成功",
    "totalPage": null,
    "totalElement": null,
    "data": null
}
